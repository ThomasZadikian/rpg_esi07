# RPG_ESI07

Projet RNCP 36286 - Expert en Informatique et Systèmes d'Information
Spécialisation: Cybersécurité

## Description
Architecture distribuée sécurisée combinant:
- Client Unity (Windows)
- API REST ASP.NET Core 8
- Portail Vue.js 3
- PostgreSQL 16

## Stack Technique
- Backend: .NET 8, PostgreSQL, Azure
- Frontend: Vue.js 3, Vite, TailwindCSS
- Client: Unity 2022.3 LTS
- Sécurité: MFA TOTP, JWT, Argon2id, TLS 1.3
